Pertanyaan Umum

Pertanyaan yang sering diajukan

Apakah EZer benar-benar gratis? expand_more
Ya. Paket gratis mencakup pelacak pengeluaran, hingga 3 anggaran, 2 tujuan tabungan, analitik, dan pengingat tagihan. Tanpa percobaan, tanpa biaya tersembunyi. Paket Plus opsional (Rp29.000/bln) membuka anggaran, tujuan, dan fitur lanjutan tanpa batas.
Apakah perlu membuat akun? expand_more
Tidak. EZer bekerja tanpa akun, email, atau login. Cukup unduh dan mulai melacak. Data Anda disimpan secara lokal di perangkat — kami tidak pernah melihatnya.
Apakah data keuangan saya aman? expand_more
Tentu saja. Semua data dienkripsi dengan AES-256 (enkripsi bank) dan hanya disimpan di perangkat Anda. Tidak ada yang diunggah ke server. Anda juga dapat mengaktifkan kunci biometrik (Face ID / sidik jari) untuk keamanan tambahan.
Apakah bekerja offline? expand_more
Ya, 100%. EZer bekerja sepenuhnya offline. Tidak diperlukan koneksi internet. Ini juga berarti data Anda tetap di perangkat dan tidak pernah berisiko dari kebocoran server.
Apa bedanya EZer dengan YNAB atau Mint? expand_more
YNAB biaya $14.99/bulan dan memerlukan pembuatan akun. Mint ditutup pada 2024. EZer gratis selamanya, bekerja offline, tidak memerlukan login, dan data Anda tidak pernah meninggalkan perangkat. Ini adalah pelacak pengeluaran paling privat.
Apa yang terjadi jika saya ganti ponsel? expand_more
Anda dapat mengekspor semua data dari EZer dan mengimpornya ke perangkat baru. Kami juga sedang mengerjakan cadangan cloud terenkripsi opsional untuk transfer data yang mulus.
