Kebijakan Privasi

Terakhir diperbarui: Maret 2025

Ringkasan: Privasi Anda adalah Prioritas Kami

🔒 EZer adalah aplikasi yang mengutamakan privasi dan offline-first. Data keuangan Anda TIDAK PERNAH meninggalkan perangkat Anda. Kami tidak mengumpulkan, menyimpan, atau menjual data pribadi Anda.

  • Semua data keuangan disimpan hanya di perangkat Anda
  • Tidak diperlukan akun atau login
  • Tidak ada cloud sync untuk data keuangan
  • Enkripsi AES-256 untuk semua data yang disimpan
  • Tidak ada pelacakan perilaku pengguna

1. Informasi yang Kami Kumpulkan

Data yang Tidak Kami Kumpulkan

EZer dirancang dengan privasi sebagai inti. Kami TIDAK mengumpulkan:

  • Data transaksi atau keuangan Anda
  • Informasi pribadi (nama, email, telepon)
  • Data lokasi
  • Kontak atau data SMS
  • Riwayat penelusuran atau penggunaan aplikasi lain

Data Analitik Anonim (Opsional)

Dengan izin Anda, kami dapat mengumpulkan data analitik anonim untuk meningkatkan aplikasi, seperti:

  • Jumlah pengguna aktif (tanpa identifikasi)
  • Fitur yang digunakan (tanpa konten)
  • Laporan crash untuk memperbaiki bug

2. Penyimpanan Data

Semua data keuangan Anda disimpan secara lokal di perangkat menggunakan SQLite dengan enkripsi SQLCipher (AES-256). Data ini tidak pernah diunggah ke server kami atau server pihak ketiga mana pun.

💾 Data Anda ada di ponsel Anda. Hanya Anda yang memiliki akses.

3. Izin Aplikasi

EZer hanya meminta izin yang diperlukan:

  • Kamera — Untuk scan struk (Plus). Foto tidak pernah diunggah.
  • Biometrik — Untuk kunci aplikasi dengan sidik jari atau Face ID.
  • Notifikasi — Untuk pengingat tagihan (opsional).
  • SMS (Android) — Untuk deteksi otomatis transaksi (opsional, dapat dimatikan).

4. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, menyewakan, atau berbagi data pribadi Anda dengan pihak ketiga. Satu-satunya layanan pihak ketiga yang digunakan adalah:

  • Firebase Analytics — Hanya data anonim jika fitur analitik diaktifkan
  • Apple/Google — Untuk pemrosesan pembayaran in-app (Plus)

5. Keamanan Data

Langkah keamanan yang kami terapkan:

  • Enkripsi AES-256 untuk semua data tersimpan
  • Kunci biometrik opsional
  • Tidak ada transmisi data keuangan melalui jaringan
  • Aplikasi bekerja sepenuhnya offline

6. Hak Anda

Anda memiliki kendali penuh atas data Anda:

  • Ekspor semua data kapan saja
  • Hapus semua data dari pengaturan aplikasi
  • Ajukan permintaan penghapusan data melalui email

Untuk permintaan penghapusan data: support@ezerapp.com

7. Hubungi Kami

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, hubungi kami:

Email: support@ezerapp.com
