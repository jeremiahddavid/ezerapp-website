1. Pengenalan

EZer ("kami", "kita") komited terhadap perlindungan privasi anda. Dasar privasi ini menerangkan bagaimana EZer mengendalikan maklumat anda apabila anda menggunakan aplikasi mudah alih kami.

Prinsip asas kami: data kewangan anda adalah milik anda sepenuhnya. Kami tidak mengumpulnya, tidak menyimpannya di pelayan kami, dan tidak berkongsinya dengan sesiapa pun.

2. Maklumat yang Dikumpul

Data kewangan — tidak dikumpul oleh kami

Semua transaksi, bajet, matlamat simpanan dan data kewangan anda disimpan secara tempatan di peranti anda sahaja. EZer tidak pernah menghantar data ini ke mana-mana pelayan.

Analitik tanpa nama (pilihan)

EZer mungkin mengumpul analitik aplikasi tanpa nama yang sangat terhad untuk membantu meningkatkan produk. Ini termasuk:

  • Bilangan pengguna aktif (tanpa identiti)
  • Ciri-ciri yang digunakan (tanpa kandungan)
  • Laporan ranap aplikasi (tanpa data pengguna)

Analitik ini tidak mengandungi sebarang data kewangan dan tidak dapat digunakan untuk mengenal pasti anda.

3. Penyimpanan Data

Semua data kewangan anda disimpan:

  • Di dalam peranti anda sahaja
  • Disulitkan dengan SQLCipher AES-256
  • Dilindungi dengan kata laluan yang dihasilkan secara selamat
  • Tidak boleh diakses dari luar peranti anda

Apabila anda memadamkan aplikasi, semua data dipadamkan sepenuhnya dari peranti anda. Tiada data tersisa di pelayan kerana tiada data yang pernah dihantar kesana.

4. Kebenaran Aplikasi

EZer mungkin meminta kebenaran berikut:

  • Pemberitahuan: Untuk peringatan bil dan peringatan bajet
  • Kamera: Untuk mengimbas resit (ciri Plus sahaja)
  • Biometrik: Untuk kunci aplikasi dengan cap jari atau ID muka

Semua kebenaran adalah pilihan. Aplikasi berfungsi sepenuhnya tanpa mana-mana daripada ini.

5. Perkongsian dengan Pihak Ketiga

EZer tidak berkongsi data kewangan anda dengan mana-mana pihak ketiga. Ini termasuk:

  • Pengiklan — tiada
  • Perkhidmatan analitik — hanya data tanpa nama
  • Rakan kongsi perniagaan — tiada
  • Pihak berkuasa kerajaan — tiada (kerana tiada data untuk dikongsi)

6. Keselamatan

Kami mengambil keselamatan dengan serius:

  • Penyulitan SQLCipher AES-256 untuk semua data kewangan
  • Tiada komunikasi rangkaian untuk data kewangan
  • Sokongan kunci biometrik (cap jari, ID muka)
  • Penilaian keselamatan berkala

7. Hak Anda

Memandangkan semua data anda disimpan secara tempatan di peranti anda, anda mempunyai kawalan penuh:

  • Akses: Semua data anda boleh diakses terus dalam aplikasi
  • Eksport: Eksport data anda pada bila-bila masa
  • Pemadaman: Padamkan semua data dari tetapan aplikasi
  • Pemindahan: Import/eksport melalui CSV
