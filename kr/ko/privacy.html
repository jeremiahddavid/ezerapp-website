개인정보처리방침

최종 업데이트: 2025년 1월 1일

🔒 핵심 원칙: 데이터 없음. 추적 없음. 서버 없음.

EZer는 재정 데이터를 수집하거나 저장하거나 전송하지 않습니다. 모든 것이 기기에 저장됩니다. 당사는 귀하의 금융 정보를 절대 볼 수 없습니다.

1. 당사가 수집하지 않는 정보

EZer는 다음을 수집하지 않습니다:

  • 거래, 예산, 저축 목표를 포함한 금융 데이터
  • 성명, 이메일 주소, 전화번호
  • 위치 데이터
  • 기기 식별자 (기본 설정)
  • 사용 데이터 또는 행동 추적

2. 기기에서의 데이터 저장

모든 재정 데이터는 오직 기기에만 저장됩니다. AES-256 암호화 (은행 수준 보안)를 사용합니다. 데이터는 인터넷 연결 없이 저장됩니다. 당사의 서버, 클라우드 스토리지 또는 제3자와 절대 공유되지 않습니다.

3. 선택적 분석

앱을 지속적으로 개선하기 위해 선택적 익명 분석을 포함합니다. 수집되는 경우, 이 데이터에는 다음이 포함됩니다:

  • 앱 충돌 보고서 (개인정보 없음)
  • 기능 사용 통계 (집계된 것만)
  • 성능 지표

이 데이터에는 절대 금융 정보나 개인 식별 정보가 포함되지 않습니다.

4. 인앱 구매

EZer Plus를 구매하는 경우 결제는 Apple App Store 또는 Google Play Store를 통해 처리됩니다. 당사는 결제 정보를 처리하거나 저장하지 않습니다. 플랫폼의 개인정보처리방침이 적용됩니다.

5. 귀하의 권리

모든 데이터는 기기에 있으므로 귀하가 완전한 통제권을 갖습니다:

  • 언제든지 앱 설정에서 모든 데이터 삭제 가능
  • 앱 삭제로 모든 데이터 완전 제거
  • 데이터 내보내기 가능

6. 어린이 개인정보 보호

EZer는 만 13세 미만 어린이로부터 의도적으로 정보를 수집하지 않습니다.

7. 이 방침의 변경

이 개인정보처리방침을 업데이트할 경우 앱 내에서 사용자에게 알려드립니다. 지속적인 사용은 업데이트된 방침에 동의하는 것으로 간주됩니다.

8. 문의하기

개인정보 보호에 대한 질문은 privacy@ezerapp.com으로 문의하세요.
